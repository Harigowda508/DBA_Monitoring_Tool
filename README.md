DBA_Monitoring_Tool is an in-house SQL Server monitoring and reporting solution that helps DBAs proactively track database health, performance, and daily operations from a single centralized platform.

Features

SQL Server health monitoring (CPU, memory, disk, database status)
SQL Agent job and backup monitoring
Performance analysis (long-running queries, blocking, deadlocks)
Index and maintenance insights
Automated data collection across all user databases
Centralized reports for DBA teams and management

Tech Stack
Microsoft SQL Server (2016+)
T-SQL (Dynamic SQL, Stored Procedures)
Web-based frontend
Windows Server deployment

Purpose
Built for internal DBA teams to automate routine checks, reduce manual monitoring, and enable faster troubleshooting.
